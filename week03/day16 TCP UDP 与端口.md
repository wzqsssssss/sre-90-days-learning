## 🔧 关键操作记录

```bash
# 查看监听端口
ss -tuln
# 或
netstat -tuln

# 模拟 TCP 连接（需安装 netcat）
nc -l 8080          # 服务端
nc 127.0.0.1 8080   # 客户端
```

---

## ⚠️ 遇到的问题 & 解法

* **问题** **：**`netstat` **命令不存在**
  **解决** **：现代 Linux 推荐用** `ss`（来自 iproute2 包），更快更准

---

## 💡 小技巧 / 收获

* **TCP：可靠、有序、有连接（三次握手）**
* **UDP：无连接、快、不可靠（适合 DNS、视频流）**
* **三次握手：SYN → SYN-ACK → ACK**
